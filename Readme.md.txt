# Retail Anomaly Detection (PoC)

## Overview
This is a proof-of-concept (PoC) for detecting anomalies in retail sales data using a simple machine learning model.

- **Model**: IsolationForest (from scikit-learn).
- **Dataset**: Dummy CSV with daily sales numbers.
- **Goal**: Identify unusual sales spikes/drops.
